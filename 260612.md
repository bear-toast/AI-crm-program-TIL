#### Intellij 주요 단축키

Alt Insert : Getter Setter

### 1. `psvm` 과 `sout` (라이브 템플릿)

- `main` 메서드 만들 때 일일이 `public static void main...` 치지 마시고, 그냥 **`psvm`** 치고 `Tab` 누르세요.
- `System.out.println()` 쓰고 싶을 땐 **`sout`** 치고 `Tab` 누르면 끝납니다.

### 2. `Ctrl + Alt + V` (변수 자동 추출)

- [](https://app.notion.com/p/180abd06af538324ab648191e77b4259?pvs=21)[https://app.notion.com/p/static-180abd06af538324ab648191e77b4259](https://app.notion.com/p/static-180abd06af538324ab648191e77b4259)예를 들어 `new ArrayList<String>();`을 먼저 치고 이 단축키를 누르면, 인텔리제이가 앞에 `List<String> list =` 부분을 자동으로 추론해서 완성해 줍니다.

### 3. `Ctrl + Alt + L` (코드 줄 맞춤)

- 코딩하다 보면 들여쓰기나 공백이 엉망이 될 때가 있죠. 이때 이 단축키 한 번만 누르면 포맷을 칼같이 아주 예쁘게 정리해 줍니다. (코딩계의 시몬스 침대랄까요...)
