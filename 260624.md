#### 요청과 응답
- Request : 서버에 보내는 요청의 종류 (CRUD) 
	- GET
	- POST
	- PATCH / PUT
	- DELETE
- Response
- Status Code : 데이터와 함께 오는 결과 알림 숫자 ('200', '404')
	- 2 성공
	- 4 내 요청 문제
	- 5 서버 문제
#### URL, JSON
- URL : 데이터가 있는 자리를 가리키는 주소
	- path parameter : /sth/1 (단일)
	- query string : /sth?num=1
- JSON : 화면과 서버가 주고받는 데이터 구조
	- Object + Array
#### fetch
- fetch('주소', {method: "POST"})

- res.json
