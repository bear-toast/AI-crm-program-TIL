## 기업 리서치 - 한화파워 (B2B MTO)
#### 메인 프로세스 (Sales & Service 위주로 고려)
①·②가 시운전→자산 등록으로 수렴하고, ③의 정비 이력이 다시 ②의 출발점이 되는 순환 구조 (DMU:Decision Making Unit 고려)

- ① 신규 설비 도입
	- 증설 계획 수립(생산 capa 부족 인지) → 요구 사양 정의 → 공급사 탐색·정보 수집(포털/레퍼런스 확인) → RFP 발행 → RFQ·견적 접수 → 기술 사양 협의 → 내부 투자 품의·승인 → 계약 체결 → 제작·납품 → 설치·시운전 → 자산(Asset) 등록·무상 보증 개시

	- 주요 관여자: 생산기획팀 실무자(메인), 공정 엔지니어(사양 검토), 구매팀(견적 비교·협상), 경영진(투자 승인)
- ② 노후화 설비 교체
	- 노후 장비 식별(정비비 증가·비계획 정지 누적) → 상급자 보고·교체 검토 착수 → 수리 vs 교체 경제성 분석 → 공급사 교체 제안 검토(자산 데이터 기반) → RFQ·견적 요청 → 호환성·다운타임 협의 → 내부 품의·승인(투자 회수 논리) → 계약 체결 → 교체 시공·시운전 → 구자산 폐기·신자산 등록

	- 주요 관여자: 설비관리팀 실무자(메인), 정비 엔지니어(이력·상태 진단), 구매팀, 공장장/경영진(승인)
- ③ 유무상 서비스
	- 서비스 요청 접수(포털/콜) → 자산·보증(Entitlement) 확인 → 유상/무상 판정 → (유상 시) 견적·승인 → 서비스 수행(정비/부품 교체) → 정비 이력 기록 → 소모성 자재 추천·주문 → 이력 누적 → 노후화 감지 시 ②번 프로세스로 연결

	- 주요 관여자: 설비관리팀 실무자(메인·②와 동일 인물), 공급사 서비스 엔지니어, 서비스 계약 담당
